# Task Manager API

This is a simple RESTful API built using **ASP.NET Core** for managing tasks.

## Features

- Create tasks
- Get all tasks
- Update tasks
- Delete tasks

## Technologies Used

- C#
- ASP.NET Core
- REST API
- Object-Oriented Programming (OOP)

## Purpose of the Project

This project was built as a backend practice project to improve my skills in:

- ASP.NET Core
- API development
- Backend architecture
- Clean code principles
